# References

This section provides comprehensive reference materials for the eBERlight Research Archive.

## Contents

| File | Description |
|------|-------------|
| [bibliography.bib](bibliography.bib) | BibTeX file for all cited papers |
| [glossary.md](glossary.md) | Synchrotron science terminology glossary |

## Usage

### BibTeX Citations

The `bibliography.bib` file contains BibTeX entries for all papers referenced throughout
this archive. To cite in LaTeX:

```latex
\bibliography{08_references/bibliography}
\bibliographystyle{unsrt}
```

### Glossary

The glossary defines technical terms used throughout the archive. Terms are organized
alphabetically and cross-referenced with the sections where they are discussed in detail.

### Links

The useful links file provides direct URLs to all external resources mentioned in the
archive, organized by category.
